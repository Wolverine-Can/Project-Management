# Project Management
A Spring Boot web application for managers to manage projects and employees.

![](demo.gif)

## Features
- PostgreSQL database with AWS RDS.
- Restful routes and services API
- Friendly and interactive UI

## Technologies

### Front-end
- Javascript, HTML, CSS
- Bootstrap 4
- Google Fonts
- Font Awesome
- Chart.js

### Back-end
- Spring Boot 2
- PostgreSQL with AWS RDS.
- Hibernate
- Thymeleaf
- Spring Security

### Deployment
- AWS EC2
